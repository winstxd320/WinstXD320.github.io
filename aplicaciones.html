Calculadora de consola.

Calculadora de consola fue uno de mis primeros proyectos escritos en el lenguaje python. Se trata de la tipica calculadora y uno de los primero programas que creamos en la universidad, cuando tratamos de operar dos variables de forma aritmetica. Todas esas operaciones las realiza este programa.

Enlaze de descarga

GITHUB -> Calculadora de consola

PyPulperiaManager

PyPulperiaManager es una aplicaciones de líneas de comando escrita en el lenguaje python, pensado para almacenar pequeñas cantidades de datos en una tabla; dirigido para los pequeños negocios en donde desean llevar sus registros en un programa pequeño y sencillo. A su vez el programa cuenta con un apartado para usuarios avanzados con conocimientos básicos en sql.

Enlaze de descarga

GITHUB -> PyPulperiaManager

Si tienes problemas a la hora de correr el código en un editor, deberas descargar las siguientes librerias con el comando PIP INSTALL: Colorama, tabulete, simpleJSON.

Tools for students

Tools for students es una aplicación grafica escrito en lenguaje visual basic. Cuenta con tres herramientas basicas: Realiza calculos aritmentos, calcula el diamentro de diferentes formas geometricas, procesador de texto.
